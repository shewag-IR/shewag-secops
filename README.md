## What This Repo Is

This is my **daily SOC practice log** — a living record
of everything I am learning and doing on the path from
zero to SOC Analyst.

It contains:

```
shewag-secops/
│
├── daily-logs/          ← Daily practice notes (what I did, what I learned)
├── l1-reports/          ← One L1 incident report per day
├── lab-screenshots/     ← Evidence of hands-on lab work
├── study-notes/
│   ├── splunk/          ← SPL queries, concepts, screenshots
│   ├── wazuh/           ← Rules, detections, agent configs
│   └── sentinel/        ← KQL queries, analytics rules, workbooks
├── weekly-reviews/      ← Sunday review summaries
└── resources.md         ← Everything useful I find along the way
```
